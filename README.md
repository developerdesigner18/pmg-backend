# pmg-media Clone API


# Technologies
* Node.js
* Express
* MongoDB 
* JSON Web Token (JWT)
* bcryptjs


# Run_Locally

Clone the project

```bash
  https://github.com/saddamarbaa/pmg-media-clone-api
```

Go to the project directory

```bash
  cd pmg-media-clone-api
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node server.js


```
